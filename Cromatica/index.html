Harmonia Cromática: Análoga

Paleta de Cores: Verde (#4CAF50), Azul (#2196F3), Amarelo (#FFEB3B)

Círculo Cromático

O círculo cromático é uma representação visual das cores, organizadas de forma circular. Ele ajuda a entender como as cores se relacionam entre si e é uma ferramenta fundamental para a criação de paletas harmoniosas.

Harmonias Cromáticas

  • Harmonia Monocromática: Utiliza variações de uma única cor.
  • Harmonia Análoga: Combina cores adjacentes no círculo cromático.
  • Harmonia Complementar: Usa cores opostas no círculo cromático.
  • Harmonia Triádica: Combina três cores equidistantes no círculo.
  • Harmonia Tetrádica: Usa duas cores complementares e suas variações.
  • Harmonia de Divisão Complementar: Combina uma cor com as duas cores adjacentes à sua complementar.

Para que Cada Harmonia Cromática

As harmonias cromáticas são utilizadas para criar diferentes emoções e atmosferas em um design. Por exemplo:

  • Monocromática: Transmite simplicidade e elegância.
  • Análoga: Cria uma sensação de harmonia e tranquilidade.
  • Complementar: Gera contraste e dinamismo.
  • Triádica: Oferece um equilíbrio vibrante.
  • Tetrádica: Proporciona complexidade e diversidade.
  • Divisão Complementar: Mantém o contraste, mas com uma abordagem mais suave.

Como Escolher as Cores dos Elementos do Site

Ao escolher as cores para os elementos do seu site, considere o seguinte:

  • Cabeçalhos: Use cores que se destaquem, mas que estejam em harmonia com o restante da paleta.
  • Parágrafos: Cores neutras são recomendadas para facilitar a leitura.
  • Bordas: Podem ser usadas para definir seções, utilizando cores que complementem o fundo.
  • Listas: Utilize cores que ajudem a organizar visualmente as informações.
  • Links: Cores que contrastem com o texto, mas que ainda estejam na paleta escolhida.
  • Tabelas: Use cores alternadas para linhas para facilitar a leitura.
